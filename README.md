# 🎮 2D Platformer

## 📖 About The Project
Author: **Ctrl-Alt-Defeat**(Hermes Hackathon 2025 Team)

This is a classic 2D platformer developed using the Unity Game Engine. This project focuses on building a robust, responsive character controller and implementing core platforming mechanics from scratch using C#. 

Whether it's precise jumping, handling environmental collisions, or managing game states, this project serves as a showcase of foundational game development concepts in a 2D space.



## ✨ Features
* **Custom Character Controller:** Smooth horizontal movement and responsive jump mechanics written in C#.
* **Physics & Collision:** Utilizing Unity's 2D Rigidbody and Collider system for accurate ground detection and wall interactions.
* **Animation States:** Seamless transitions between idle, running, jumping, and falling animations.
* **Modular Codebase:** Clean and scalable scripts designed for easy expansion of game features.

## 🛠️ Built With
* **Engine:** [Unity](https://unity.com/) (2D Core)
* **Language:** C#
* **Version Control:** Git & GitHub
